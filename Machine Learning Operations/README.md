Project: PredictStroke – Stroke Risk Prediction System

Description:
PredictStroke is a machine learning–based project aimed at predicting stroke risk using biological and social factors. The project focuses on data preprocessing, model training, evaluation, and deployment within a web-based application to support both specialists and non-expert users.

Software & Tools:
Python, Streamlit (web application), Git/GitHub (version control), Scikit-learn

Programming Language:
Python

Models & Techniques Used:
Logistic Regression, Decision Tree, Random Forest, Support Vector Machine (SVM); data preprocessing, oversampling techniques (SMOTE/oversampling), K-Fold Cross Validation.
